Список событий сеанса Оператор

По этим событиям можно отследить ход исполнения Процедуры, если события выводятся в лог Оператор.


++ События начала сеанса
С1 Начало сеанса, Начата процедура инициализации сеанса.
С2 Начало сеанса, Завершена процедура инициализации сеанса 
С3 Ошибка при инициализации сеанса
- критическая ошибка, невозможно продолжить работу Оператор.

++ Основные события исполнения запроса
С10 Поступил новый запрос от пользователя.
С11 Исполнение запроса успешно завершено.
С12 Исполнение запроса прервано/отменено пользователем.
 - возникает, если:
   - пользователь нажал Ctrl+C или подобную Break комбинацию клавиш.
   - пользователь выбрал вариант ответа Отмена в диалоге Процедуры.
 - уже произведенные изменения не откатываются назад автоматически.
   Пользователь должен сам с ними разбираться.
С13 Критическая ошибка при исполнении запроса.
 - возникает при исключении в процессе исполнения запроса и сигнализирует
    о существенной неисправности в системе Оператор, 
    не позволяющей продолжить работу Оператор.
 - уже произведенные изменения не откатываются назад автоматически.
   Пользователь должен сам с ними разбираться.
С14 Некритическая ошибка при исполнении запроса.
 - возникает при ошибке внутри исполняемой Процедуры и сигнализирует
    о невозможности исполнения текущего запроса.
 - уже произведенные изменения не откатываются назад автоматически.
   Пользователь должен сам с ними разбираться.

++ События завершения сеанса
