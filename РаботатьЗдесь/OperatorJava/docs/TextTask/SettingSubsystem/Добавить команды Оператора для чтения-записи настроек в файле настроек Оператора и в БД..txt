Добавить команды для  настроек в ФайлНастроекОператора.
- добавить настройку в файл
- удалить настройку из файла
- просмотреть список настроек файла
- редактировать одну настройку по ее названию.

Такие же команды нужны для управления Настройками в ТаблицеНастроекОператора. 

При создании Настройки пользователь должен будет указать для нее название и комментарий.
Оператор должен предупредить пользователя, если настройка с таким названием уже имеется в файле настроек.
Только предупредить, так как формат настроек допускает существование нескольких настроек под одним названием.
