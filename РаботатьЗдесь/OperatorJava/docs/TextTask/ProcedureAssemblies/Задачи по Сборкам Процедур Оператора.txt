+ Сборка Процедур Оператора должна иметь номер версии Движка Оператора, для которого она создана.
- Проверять для каждой сборки Процедур номер версии Движка Оператора.
  - При загрузке менеджера
  Если номер версии не совпадает, сообщить о несовпадении версий, но не загружать и не исполнять процедуру из сборки.
  
- Добавить команды для просмотра списка имеющихся Сборок Процедур и процедур в них через интерфейс Оператора.
  Предполагается помечать сборки, классы и методы атрибутом - аннотацией OperatorProcedure, 
  содержащей состояние, название и описание Процедуры. 
   Вот по этим данным пользователь может поискать подходящую процедуру для команды.
